*HTML Blocks

- Here we have inline Html elements:
An inline element does not start on a new line and only takes up as much width as necessary.
Span is an example of a inline element which is used to Define an inline section.

and Block Html elements: A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

Div tag is a Block level element and is used to Define a block element or to group of more block-elements
